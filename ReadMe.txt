features:

Visualize velocity, scalar, divergence, vorticity, streamline ...
save image
print sensor data (not available yet)


control:

left hold   : rotation
right hold  : pan (translate)
middle hold : scale
left click  : print particle information



shortcut:

space    : reset camera
ctrl + p : print current image
